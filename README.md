# SmartCitizenHub
An AI-powered smart city platform enabling intelligent civic issue reporting, predictive waste management, crowd analytics, and real-time public transport insights for efficient urban governance.

## Stack
- React (Vite) + React Router + Tailwind CSS (client)
- Node.js + Express + MongoDB (server)

## Setup

### 1) Server
1. Create a .env file in server/ (use server/.env.example as a template).
2. Install dependencies and start the server.

### 2) Client
1. Install dependencies and start the client.

## Scripts

### Server
- npm install
- npm run dev

### Client
- npm install
- npm run dev

## Notes
- The API base URL is proxied in the client to /api (see client/vite.config.js).
- Make sure MongoDB is running and MONGODB_URI is set.
